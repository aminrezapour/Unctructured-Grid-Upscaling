GPRS

GPRS is a general purpose research simulator developed and maintained by Stanford University's Reservoir Simulation (SUPRI-B) Industrial Affiliates Program. GPRS is a state-of-the-art object oriented reservoir simulator written in C++.

GPRS is based on a generalized compositional formulation and offers the following advantages:

  • Structured or unstructured grid
  • Arbitrary choice of primary variables
  • Adaptive implicit formulation (FIM, IMPES, IMPSAT)
  • Black-oil or compositional
  • Two-point or multi-point discretization schemes
  • Multi-stage preconditioners with different linear solvers
  • Advanced well modeling (multi-segments with chokes, drift-flux model)
  • Thermal extension
  • Chemical reaction

GPRS Release Package

GPRS release package consists of three parts. The first part is the documents and enhancements. The “documents” folder includes the user manual and all development reports, Ph.D. dissertation, M.S. report and application programming interfaces (APIs), see the description file. The “enhancements” folder includes all enhancement features and extensions of GPRS. In each sub-folder, the samples and report(s) are provided for each special enhancement or extension, see description file.

The second part includes the code, libraries, executables, and build instruction for each individual development environment. There is also a html (for Windows version) or text (for Linux or Unix version) file which describes the contents of this part.

The third part is the samples which are divided into 2P (two point flux) and MP (multi-point flux).
